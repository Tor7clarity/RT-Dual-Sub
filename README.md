安装方法:


由于这个Mod是直接替换原始字幕文件，建议先备份原版字幕.json文件（当然也可以通过平台校验完整性恢复）。
1. 下载并解压本Mod至 _Warhammer 40,000 Rogue Trader\WH40KRT_Data\StreamingAssets\Localization_ (Steam库：游戏页面 管理>浏览本地文件）
2. 运行RT Dual Sub，根据屏幕提示输入数字，选择显示在前的主语言与副语言（括号内显示）,工具会即时在当前目录生成一个名为 merged_xxxx_xxxx.json 的新文件。
3. 备份原版语言文件，然后将新生成的文件重命名并覆盖原本游戏内所选语言的字幕文件（如改为 zhCN.json）即可。


How to install：

1. Download and extract this mod into the game's localization directory: _Warhammer 40,000 Rogue Trader\WH40KRT_Data\StreamingAssets\Localization _
2. Run RT Dual Sub.exe, enter the corresponding numbers to choose your Primary (displayed first) and Secondary (inside parentheses) languages. A new file named merged_xxxx_xxxx.json will be generated accordingly.
3. Backup your original subtitle files, then rename the newly generated file to replace the original (e.g., rename it to enGB.json).
